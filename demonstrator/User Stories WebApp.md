# User Stories HTML 5 Blueprint Demonstrator App

## Generell

### User Story GE1: Login

Als ein Blipper möchte ich mich mit meiner Telefonnummer und meinem Passwort gegen das System authentifizieren können, damit niemand anderes in meinem Namen blippen kann.


### User Story G2: Kontakt auswählen

Als ein Blipper möchte ich aus meinen Kontakten einen Kontakt auswählen können, damit ich ihn auf der Karte finden kann.


### User Story G3: Kontakte auf Karte anzeigen

Als ein Blipper möchte ich Position und Zeitpunkt der jeweils letzten Blipz meiner Kontakte angezeigt bekommen, damit ich mir einen Überblick über mein Netzwerk verschaffen kann.



## Webapplikation

### User Story WA1

Als Geschäftsreisender möchte ich meine Position morgen blippen können, damit ich Abends mit Kollegen, die in der Nähe sind mit mir ein Bier trinken gehen können.


### User Story WA2

Als Geschäftsreisender möchte ich wissen, wer sich ab morgen für meine geplantes Ziel ebenfalls angekündigt hat, um mit diesen Kollegen ein Bier trinken gehen zu können.


### User Story WA3

Als Geschäftsreisender möchte ich wissen, wer zu meinem Reisetermin dasselbe Ziel wie ich hat, damit ich mit diesen Personen evtl. eine Fahrgemeinschaft bilden kann.


### User Story WA4

Als Clubgänger möchte ich blippen, in welchem Club ich in 2 Stunden abtanzen werde, damit meine Kontake die Gelegenheit haben, den Laden mit mir gemeinsam in Schutt und Asche zu legen.


### User Story WA5

Als ein Benutzer der Blip Webapplikation möchte ich über neue Blipz meiner Kontakte automatisch durch Anzeige auf der Karte und akustisch informiert werden, damit ich dies auch mitbekommen, wenn ich arbeite.


### User Story WA6

Als ein Benutzer der Webapplikation möchte ich eine Liste der letzten 42 Blipz meiner Kontakte sehen, damit ich keinen Blip verpasse, selbst wenn der Browser aktuell nicht geöffnet ist.


### User Story WA7

Als Gastegeber eines Diners möchte ich nur die Blipz der letzten 30 Minuten sehen, damit ich weiss, wann ich ungefähr mit meinen Gästen zu rechnen habe und das Soufflée rechtzeitig in den Ofen schieben kann.


### User Story WA8

Als Gastgeber eines Diners möchte ich eine Blip-Anfrage an alle meine Gäste senden können, damit ich vorab über deren voraussichtliche Ankunft informiert werden kann.
